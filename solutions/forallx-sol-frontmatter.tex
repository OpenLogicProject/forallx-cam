\noindent This booklet contains model answers to the practice exercises found in \forallxcam. For several of the questions, there are multiple correct possible answers; in each case, this booklet contains at most one answer.  Answers are given in blue; please contact Tim Button at \url{http://www.nottub.com/forallx.shtml} if you have accessibility requirements.
